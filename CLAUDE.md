# Robotics Intelligence Dashboard

This is a Next.js 14 dashboard for tracking the robotics and physical AI industry.

## Key Features
- Interactive 3D globe with company markers
- Robotics Narrative Index (composite sentiment score)
- Funding tracker with NewsAPI integration
- Government contracts data (USASpending.gov)
- AI chat assistant (GPT-4 powered)

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Three.js / React Three Fiber (globe)
- Tailwind CSS
- Zustand (state management)

## Development
```bash
npm run dev -- -p 3001
```

## Environment Variables
See `.env.example` for required API keys.
